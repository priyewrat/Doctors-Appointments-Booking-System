# Doctors Appointments Booking System

A full‑stack **MERN** application for managing doctor appointments.  
This project includes three main modules:

- **Frontend** – Patient-facing interface for booking appointments
- **Admin Dashboard** – Admin panel for managing doctors, users, and appointments
- **Backend** – RESTful API with authentication, scheduling, and database integration

---

## 🚀 Tech Stack
- **MongoDB** – Database
- **Express.js** – Backend framework
- **React.js (Vite)** – Frontend framework
- **Node.js** – Runtime environment
- **Cloudinary** – Image storage
- **JWT Authentication** – Secure login and session handling

---

## ⚙️ Installation & Setup

1. Frontend
cd frontend
npm install
npm run dev

2. Admin
cd admin
npm install
npm run dev

3. Backend
cd backend
npm install
npm run server

📌 Features
👩‍💻 User Module
User registration & login

Browse doctors and book appointments

View appointments in "My Appointments"

Cancel appointments directly from "My Appointments" dashboard

👨‍⚕️ Doctor Module
Doctor login & secure authentication

Doctor dashboard for managing appointments

Add convenient slots for availability

Cancel or mark appointments as completed

View latest appointments

Dashboard metrics: total earnings, total appointments, total patients

🛡️ Admin Module
Admin dashboard for managing doctors, users, and bookings

Add new doctors

Mark doctors as available/unavailable

Delete patients if required

Monitor system activity and earnings
